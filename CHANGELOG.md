# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-04-27

### Added
- Initial release
- `unused` command: detect CSS selectors with no matching HTML elements
- `colors` command: extract and normalize color values from CSS
- `specificity` command: audit selector specificity scores
- Pretty, table, and JSON output formats
- Programmatic API exports
- Full test suite with 80%+ coverage
